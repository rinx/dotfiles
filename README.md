## Usage


### Deploy

To deploy assets in the home directory, put this repository into `~/.dotfiles` and run `make deploy`.

```sh
git clone https://github.com/rinx/dotfiles.git ~/.dotfiles
```

```sh
cd ~/.dotfiles
make deploy
```
